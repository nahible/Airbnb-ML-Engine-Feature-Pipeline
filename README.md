# Airbnb-ML-Engine-Feature-Pipeline
Engineered an end-to-end ML pipeline using Scikit-Learn to predict Airbnb Superhost status. • Optimized models via GridSearchCV tuning, evaluating results using ROC-AUC curves. • Applied SelectKBest feature selection to isolate top 10 signals, achieving 0.804 AUC. • Serialized the optimal model using Pickle for seamless deployment
